# Prism insurance Claims Analysis using Power BI Project06
📌 Project Overview
  The Prism Insurance Analytics Dashboard is an interactive Power BI project designed to analyze insurance policies, claims, premiums, and customer-related data.     The dashboard provides meaningful business insights through dynamic visualizations, helping stakeholders monitor claim performance, policy activity, and overall    insurance operations.
  This project transforms raw insurance data into an easy-to-understand analytical report for better decision-making and operational efficiency.

📊 Key Features
    1.Interactive Power BI dashboard with slicers and filters
    2.Analysis of insurance claims by status (Rejected, Settled, Pending)
    3.Premium amount and coverage amount tracking
    4.Policy type analysis (Travel, Health, Auto, Life, Home)
    5.Active vs Inactive policy monitoring
    6.Claim amount analysis by age group
    7.Detailed customer and policy data table
    8.KPI cards for quick business insights.
    
 📈 Dashboard Insights
    The dashboard helps users:
      1.Track total premium, coverage, and claim amounts
      2.Monitor claim settlement trends
      3.Identify high-performing policy categories
      4.Analyze customer demographics and age-based claim distribution
      5.Evaluate active and inactive insurance policies
      6.Improve insurance business reporting and decision-making.
      
 🛠️ Tools & Technologies Used:
    1.Power BI
    2.Microsoft Excel / CSV Dataset
    3.DAX (Data Analysis Expressions)
    4.Data Cleaning & Transformation
    5.Data Visualization.

 📂 Dataset Information
    The dataset contains insurance-related information such as:
     1.Policy Number
     2.Customer ID
     3.Claim Number
     4.Age & Gender
     5.Coverage Amount
     6.Premium Amount
     7.Policy Type
     8.Claim Status
     9.Claim Date
     10.Policy Start & End Dates
      
  📌 KPIs Included:
      1.Total Premium Amount
      2.Total Coverage Amount
      3.Total Claim Amount
      4.Claim Status Distribution
      5.Policy Activity Status
      6.Age Group Claim Analysis.
